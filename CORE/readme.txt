System Requirement:

windows7:  64-bit operating system
jdk 1.8 installed
Maven installed
Eclipse or other IDE installed


For Eclipse:
1.from Command Prompt, go to CORE folder and run "mvn clean install" command
2 from Command Prompt, go to CORE folder and run "mvn eclipse:clean eclipse:eclipse" command
3.import CORE project in Eclipse
4.run the main class CoreServer
